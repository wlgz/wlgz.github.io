css各种居中

总结了一些css中常见的关于居中的问题

  1. 子盒子居中问题

    1. 想要让一个子盒子在父盒子中居中显示，给父子盒子都设置定位，子绝父相，再给子盒子设置：top:50% left:50% 让子盒子走父盒子的一半，然后在让子盒子走自己宽高的一半，通过margin-left、margin-right.但是这种方法局限性强，不能灵活写数值.
    2. 利用css3中伸缩盒子的属性：display:flex.在给子盒子设置margin:auto.子盒子自动居中，随意改变盒子的值都可以，这个方法比较灵活。
    3. 利用css3中transform这个属性，同1：父子盒子都设置定位，子绝父相， 再给子盒子设置：top:50% left:50%,
      在给子盒子设置transform:translate(-50%,-50%)，利用这个属性让子盒子走自己宽高的一半。

git基础命令

  • 因为git是基于node的工具，所以需要先在cmd中查看安装了node npm没有，通过命令 node -v和 npm -v查看；
  • pwd 查看当前工作目录
  • git clone +c仓库地址 克隆一个仓库
  • git init 初始化一个仓库
  • git status 查看当前工作树状态
  • git add 文件名 将操作文件添加到暂存区
  • git add -A 将全部修改的文件全部提交到暂存区
  • git commit -m ‘提交日志’ 添加提交日志并提交到本地仓库
  • git log 查看提交日志
  • git push 仓库地址 分支 将本地仓库的相应分支文件添加到远程仓库
  • git pull 仓库地址 分支 将远程仓库相应的文件从远程仓库拿下来
  • git branch 查看分支
  • git branch 名字 创建相应的分支
  • git checkout 名字 切换到相应的分支
  • git config –global user.name ‘用户名’ 设置用户名
  • git config –global user.email ‘邮箱’ 设置邮箱
